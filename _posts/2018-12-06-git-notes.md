**安裝 Git 在 Ubuntu上**

```
apt-get update
apt-get install -y git
```

**常用的 Git alies**

```
alias gitb='git branch -a'
alias gitl='git log --graph --oneline --decorate'
alias gitla='git log --graph --oneline --decorate --all'
alias gits='git status -s'
```
